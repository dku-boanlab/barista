plain 802.11 linktype. Every packet in the buffer contains the raw 802.11 frame, including MAC FCS.

AIRPCAP_LT_802_11_PLUS_RADIO 

802.11 plus radiotap linktype. Every packet in the buffer contains a radiotap header followed by the 802.11 frame. MAC FCS is included.

Unknown linktype. You should see it only in case of error.
